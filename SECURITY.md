# Security Policy

## Supported Versions

Currently, only the latest version of the application is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within K-Guard, please open an issue with the label `security` or contact the maintainer directly via GitHub (using my pseudo). 
All security vulnerabilities will be promptly addressed.

## Disclaimer

**Educational & Portfolio Purpose:** K-Guard is developed as a Proof of Concept (PoC) and portfolio project demonstrating operational infrastructure security and DevSecOps practices. While efforts are made to ensure security, it is provided "as is" without warranty of any kind. Users deploying this in a production environment should perform their own complete security audit.
